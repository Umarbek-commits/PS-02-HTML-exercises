Section 2 — Assignment 7

Bracket Balance

Условие

Проверьте, правильна ли скобочная последовательность, содержащая ()[]{}.

Пример

Input: ({[]})
Output: YES

Решение (Python)

Код:

s=input().strip()
stack=[]
pairs={')':'(',']':'[','}':'{'}
for ch in s:
    if ch in '([{': stack.append(ch)
    elif ch in ')]}':
        if not stack or stack.pop()!=pairs[ch]: print('NO'); break
else:
    print('YES' if not stack else 'NO')

Сложность: O(n)
