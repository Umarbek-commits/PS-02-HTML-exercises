Section 1 — Assignment 3

Check Prime

Условие

Проверьте, является ли число простым.

Пример

Input: 17
Output: YES

Решение (Python)

Код:

n = int(input())
if n < 2:
    print('NO')
else:
    i = 2
    while i * i <= n:
        if n % i == 0:
            print('NO'); break
        i += 1
    else:
        print('YES')

Сложность: O(sqrt(n))
