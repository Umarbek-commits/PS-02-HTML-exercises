Section 2 — Assignment 19

Validate IPv4

Условие

Проверьте, является ли строка корректным IPv4 адресом.

Пример

Input: 192.168.0.1
Output: YES

Решение (Python)

Код:

s=input().strip()
parts=s.split('.')
if len(parts)!=4: print('NO')
else:
    ok=True
    for p in parts:
        if not p.isdigit() or not 0<=int(p)<=255 or (p!='0' and p[0]=='0'):
            ok=False; break
    print('YES' if ok else 'NO')

Сложность: O(1)
