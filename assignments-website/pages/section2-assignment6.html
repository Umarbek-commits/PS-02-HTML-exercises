Section 2 — Assignment 6

Longest Common Prefix

Условие

Найдите наибольший общий префикс среди списка строк.

Пример

Input: flower flow flight
Output: fl

Решение (Python)

Код:

arr=input().split()
if not arr: print('')
else:
    pref=arr[0]
    for s in arr[1:]:
        while not s.startswith(pref):
            pref=pref[:-1]
            if not pref: break
    print(pref)

Сложность: O(S) where S total chars
