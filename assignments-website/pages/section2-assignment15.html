Section 2 — Assignment 15

Dijkstra (small)

Условие

Найдите кратчайшие пути от вершины 0 в взвешенном графе (n≤1000) — простой Dijkstra.

Пример

Input:\n4 4\n0 1 1\n1 2 2\n0 2 4\n2 3 1\nOutput: 0 1 3 4

Решение (Python)

Код:

import heapq
n,m=map(int,input().split())
g=[[] for _ in range(n)]
for _ in range(m):
    u,v,w=map(int,input().split()); g[u].append((v,w))
d=[10**18]*n; d[0]=0
pq=[(0,0)]
while pq:
    dist,u=heapq.heappop(pq)
    if dist!=d[u]: continue
    for v,w in g[u]:
        if d[v]>dist+w:
            d[v]=dist+w; heapq.heappush(pq,(d[v],v))
print(' '.join(str(x if x<10**17 else -1) for x in d))

Сложность: O((n+m) log n)
